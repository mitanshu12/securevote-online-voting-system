SecureVote - Online Voting System

SecureVote is a secure online voting system built using Python and Django.  
It allows users to register, login, cast votes securely, and view election results.

 Features
- User registration and login authentication
- Secure vote casting
- Admin panel for managing elections
- Automatic vote counting and result generation
  
Tech Stack
- Python
- Django
- SQL Database
- HTML/CSS

Project Purpose
This project demonstrates how secure digital voting systems can be developed using web technologies.
